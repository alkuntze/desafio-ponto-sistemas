[TÍTULO]
Desafio Ponto Sistemas.

[DESCRIÇÃO]
Sistema desenvolvido conforme especificação, para o cadsatro de usuário, 
contendo os campos NOME, EMAIL e IDADE.
O projeto foi desenvolvido em HTML5, CSS3, JavaScript e PHP 7.4.
Ferramenta usada para o desenvolvido: 
	- NetBeans 8.2
	- FireFox 74.0 (64-bit)
	- PHPMyAdmin 4.9.2


[PRÉ-REQUISITOS]
- Servidor de arquivos - Apache 2.0 
    versão: Apache/2.4.41
- PHP Version 7.4.1 
- Módulo de accesso a Banco de dados PDO habilitado
- Servidor de Banco de Dados - 
    MariaDB - 10.4.11

[GUIA DE INSTALAÇÃO]
1. - Descompactar os arquivos no diretório do servidor de páginas.

2. - A Estrutura de Diretórios deve ficar como a seguinte:
    DOCUMENTACAO
        |+desafio_pontosistemas.sql
        |+README.txt
    imagens
        |+logomarcaPontoSistemas.png
    include
       |+_css
            |+estilo.css
       |+_fonts
            |+bubblegum-sans-regular.otf
            |+computer-font.ttf
       |+_js
            |+functions.js
    +Conn.php
    +Usuario.php
    +apagar.php
    +cadastrar.php
    +cadastrar_cgi.php
    +editar.php
    +editar_cgi.php
    +footer.php
    +header.php
    +index.php
    +visualizar.php

3. - Criar a base de dados contida no script SQL que está no diretório:
    DOCUMENTAÇÃO
        |_desafio_pontosistemas.sql

4. - Abrir o arquivo Conn.php e alterar os dados de conexão com o banco de dados.
